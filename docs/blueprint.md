# **App Name**: Ascend Wealth

## Core Features:

- Role-Based Dashboard: Displays a dashboard tailored to the user's role (Super-Admin, Admin, Associate, Customer), showing relevant metrics and data visualizations as per specifications.
- Hierarchical User Management: Enables Super-Admins to manage Admins, Admins to manage Associates, and Associates to manage Customers, reflecting the organizational hierarchy.
- Secure Impersonation: Allows users with sufficient privileges (Super-Admin, Admin) to impersonate lower-level users for support and management, with strict adherence to the defined rules and backend enforcement.
- Family Account Management: Enables Customers to manage their family members within the platform, including adding, editing, and deleting family member profiles.
- Document Vault: Provides a secure document storage solution for Customers to upload and manage important financial documents, organized by family member and category.
- Role and Permission Management: Allows Super-Admins to create, edit, and delete roles with associated permissions, ensuring fine-grained control over user access and capabilities. This tool can ensure no roles are deleted when there is activity connected to that role.
- Data Visualization Tool: Provides a tool to the application that leverages the AI to detect when a change in user assets should be presented as a chart, instead of other textual options.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a premium and trustworthy feel, reflecting the importance of financial data. It will stand out clearly against the dark background.
- Background color: Dark gray (#303030) for a Gen-Z modern UI aesthetic, providing contrast and focus to the content.
- Accent color: Electric purple (#BE4BEE), an analogous color used sparingly to guide the user's eye, to call attention to important details without being distracting.
- Body font: 'Inter', a sans-serif for a modern and clean reading experience.
- Headline font: 'Space Grotesk', a sans-serif used for headings, bold but inviting. 
- Use clean, minimalist icons to represent different financial categories and actions, ensuring clarity and ease of use.
- Implement a clear and structured layout with a role-aware sidebar navigation for easy access to relevant sections.
- Incorporate subtle transitions and animations to enhance user experience, such as loading indicators and page transitions, without being distracting.
